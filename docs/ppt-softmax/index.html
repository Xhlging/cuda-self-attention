THE FORMULA

Softmax 是什么？

Safe 版本: 所有值先减最大值再做 softmax，防止 e^{100} 溢出
直观理解

把一组任意实数 → 转成一组概率

大数 → 高概率，小数 → 低概率
全部加起来 = 1

softmax([2, 1, -1])
= [0.67, 0.24, 0.09]
最大的 2 → 67% 概率

THE PROBLEM

为什么要用 GPU 算 Softmax？

场景

大语言模型（ChatGPT）
每一层、每个 token
都要算一次 Softmax

32,768 个 token
每个 token 4,096 维
= 32,768 次 / 推理

GPU 方案

每个线程独立算一行
几千行 → 几千个线程
同时并行计算

核心问题: 同一行内
怎么让 32 线程协作？

NVIDIA GPU 硬件固定 32 线程 = 1 Warp
（从 2012 年 Kepler 到 2025 年 Blackwell 从未变过）
Warp 内线程可以单周期交换数据——最快协作方式

NAIVE · 04
04 / 10
VERSION 1

Naive Softmax: 每个线程单干

// 线程 tid 独立计算第 row 行的 softmax
// 第一遍: 找最大值 (Safe Softmax)
max_val = x[0];
for (i = 1; i < D; i++)
    max_val = fmaxf(max_val, x[i]);

// 第二遍: 算 exp 并求和
sum_val = 0;
for (i = 0; i < D; i++) {
    y[i] = __expf(x[i] - max_val);
    sum_val += y[i];
}

// 第三遍: 归一化
inv_sum = 1.0f / sum_val;
for (i = 0; i < D; i++)
    y[i] *= inv_sum;
问题在哪？

每个线程串行遍历 D 个元素三次。

D=1024 时，每线程 3072 次操作。
线程之间完全没有协作。

Grid: [rows/256]
Block: [256]
策略: 每线程串行三遍
WARP · 05
05 / 10
KEY CONCEPT

Warp: GPU 的最小调度单位

什么是 Warp？

32 条线程编成一组
称为一个 Warp
GPU 以 Warp 为单位调度执行

线程 0 ─┬─ Warp 0
线程 1 ─┤
⋮ │
线程 31 ─┘
Warp 的特殊能力

__shfl_down_sync
同 Warp 内的线程可以
直接交换寄存器数据
不需要共享内存！

线程 i 收到 i+offset 的值
蝴蝶归约 → log₂(32) = 5 步
32 个值 → 合并为 1 个

REDUCE · 07
07 / 10
REDUCTION + BROADCAST

归约产生在 lane 0，广播分发给所有人

Step A: 蝴蝶归约 (Reduce)
// 每轮折半: 线程 i 收到线程 i+offset 的值并合并
for (int offset = 16; offset > 0; offset /= 2)
    sum_val += __shfl_down_sync(
        0xFFFFFFFF, sum_val, offset);
offset=16
lane 0←16, 1←17, ..., 15←31 → 剩 16 个
offset=8
lane 0←8, 1←9, ..., 7←15 → 剩 8 个
offset=4
lane 0←4, 1←5, 2←6, 3←7 → 剩 4 个
offset=2
lane 0←2, 1←3 → 剩 2 个
offset=1
lane 0←1 → 只在 lane 0
Step B: 广播 (Broadcast)
// 把 lane 0 的值复制给 Warp 内所有 32 条线程
sum_val = __shfl_sync(
    0xFFFFFFFF, sum_val, 0);  // src_lane=0
两步缺一不可

归约 把 32 个值合并为 1 个（落在 lane 0）
广播 把这 1 个值复制给人人（所有 lane 都拿到）

WARP SOFTMAX KERNEL

完整代码: 35 行实现 Warp Softmax

// Grid: [rows], Block: [32] — 一行数据恰好一个 Warp 处理
template <int D>
__global__ void softmax_warp(
    const float* __restrict__ input,
    float* __restrict__ output,
    int rows, int stride)
{
    int row  = blockIdx.x;              // 一行一个 Block
    int lane = threadIdx.x;             // Warp 内编号 (0~31)
    if (row >= rows) return;
    const float* x = input + row * stride;
    float* y = output + row * stride;

    // ===== Step 1: 跨步访问找局部最大值 =====
    float max_val = -INFINITY;
    for (int i = lane; i < D; i += 32)
        max_val = fmaxf(max_val, x[i]);

    // ===== Step 2: 蝴蝶归约 + 广播 =====
    for (int offset = 16; offset > 0; offset /= 2)
        max_val = fmaxf(max_val,
            __shfl_down_sync(0xFFFFFFFF, max_val, offset));
    max_val = __shfl_sync(0xFFFFFFFF, max_val, 0);

    // ===== Step 3: 算 exp + 归约求和 + 广播 =====
    float sum_val = 0.0f;
    for (int i = lane; i < D; i += 32) {
        float e = __expf(x[i] - max_val);
        y[i] = e;  sum_val += e;
    }
    for (int offset = 16; offset > 0; offset /= 2)
        sum_val += __shfl_down_sync(0xFFFFFFFF, sum_val, offset);
    sum_val = __shfl_sync(0xFFFFFFFF, sum_val, 0);

    // ===== Step 4: 归一化输出 =====
    float inv_sum = 1.0f / sum_val;
    for (int i = lane; i < D; i += 32)
        y[i] *= inv_sum;
}
